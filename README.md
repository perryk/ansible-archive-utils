archive-utils
=========

Ansible role that installs common archive utilities on Linux.

Requirements
------------

You just need a target host that Ansible can control.

Role Variables
--------------

See vars.

Dependencies
------------

None.

Example Playbook
----------------

```yaml
- hosts: servers
  roles:
     - { role: devoperate.archive-utils }
```
